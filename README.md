### Git Repo for Paper


Binary Dependencies:
  * pdflatex
  * bibtex
  
Library Dependencies:
  * amsmath
  * amsthm
  * amsfonts
  * verbatim
  * mathtools
  * float
  * graphicx
  * xcolor
  * subfig

Build instructions:

  ````$ git clone https://github.com/efrubin/thesis-writeup.git````
  
  ````$ ./buildpaper rebuild main````

To get updates, go to the thesis-writeup directory and use

  ````$ git pull````
